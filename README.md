## Getting Started

Here are the steps to run the application.

### Prerequisites

This is things you need to use the software and how to install them.

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation & Run App

Here are the steps to run the application

1. Extract File
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run on dev env
   ```js
   npm run dev
   ```
